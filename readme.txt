== Description ==

Product Badges is a simple extension for Easy Digital Downloads which allows site operators to add 'badges' to products on their EDD-powered website.

== Installation ==

1. Upload `edd-product-badges` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your product badges through the new meta box on the Add and Edit
Downloads pages.
